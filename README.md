## 📋 项目介绍

ProxyCC 是一个专注于 AI 大模型 API 中转服务的**导航对比平台**，为国内开发者解决以下痛点：

- ❌ 官方 API 国内访问困难
- ❌ 中转平台众多，难以选择
- ❌ 价格不透明，容易被坑
- ❌ 配置教程分散，找不到靠谱方案

**我们提供：**
- ✅ 主流中转平台一站式对比
- ✅ 实时价格、稳定性评测
- ✅ 详细接入教程（Claude Code / OpenAI / Gemini）
- ✅ 直达注册，快速上手
